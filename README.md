# BDS Deep Learning Assignments

This Github Repository is used for weekly assignments in the Deep Learning module at Business Data Science at Aalborg University. These weekly assignments are all made by Alexander Christiansen, ge48ab@student.aau.dk.

Each assignment will have its own folder with the no. of the assignment. The structure below shows what the name/topic is for each assignment.

**Repository Structure:**

```
BDS-Deep-Learning-Assignments/
│
├── Assignment1/
│   └── SGD Mechanics & Attention Context
├── Assignment2/
│   └── Green Patent Detection (PatentSBERTa)
├── Assignment3/
│   └── Green Patent Detection: Advanced Architectures (Agents vs. QLoRA)
│
└── README.md
```

Each assignment will be submitted through mail with a link to the Git Repo and a link to a technical video.
